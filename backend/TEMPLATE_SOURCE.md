This directory is the Harness-level RuoYi fullstack foundation for projects
that select `backend_profile = "ruoyi-springboot3"`. It includes the backend
modules and `ruoyi-ui`. Preserve the upstream `LICENSE`.

Copy and tailor this template during a project's infrastructure task. SDD
should generate business increment code inside the RuoYi conventions instead
of rebuilding the management framework.
Do not edit a project's `vendor/` reference source as business code.

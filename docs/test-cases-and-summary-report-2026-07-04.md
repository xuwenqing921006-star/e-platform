## 七、当前遗留问题清单

| 优先级 | 问题 | 影响 | 建议 |
| --- | --- | --- | --- |
| P1 | `/h5` 无尾斜杠返回 404 | 用户直接输入无尾斜杠地址会打不开 | Nginx 增加 `/h5` 到 `/h5/` 的 301 |
| P1 | 后台子路由直达/刷新存在回登录风险 | 用户刷新内容管理/产品管理可能被打断 | 复核后台 history fallback、路由守卫、token 存储 |
| P1 | H5 多栏目无正式数据 | 用户看到大量空态 | 正式上线前补齐每个栏目真实内容 |
| P2 | SAFE 外链不可达 | 用户点击 SAFE 失败 | 确认官方地址；增加复制链接/稍后再试兜底 |
| P2 | 根路径 `/api/**`、`/prod-api/**` 404 | 若后续构建 API base 改错会出问题 | 部署文档和 Nginx 明确 `/h5/api`、`/admin/prod-api` |
| P2 | 产品详情长正文偏密 | 移动端阅读体验一般 | 优化行高、段落距、长文本换行 |
| P2 | 附件功能线上缺少样本复测 | 附件下载真实链路未用线上数据覆盖 | 新增一篇带附件文章复测 |
| P3 | 图片缩放手势未真机自动化 | 仅静态确认代码逻辑 | 使用手机微信/浏览器真机复测 |
| P3 | 导入提交未在线上执行 | 避免重复数据，未覆盖 commit | 准备测试数据和回滚方案后执行 |

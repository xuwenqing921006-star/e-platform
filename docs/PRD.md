# 央行 E 平台 PRD

> 状态：阶段 A 初稿。页面清单已确认，功能定义、版本规划、关键链路和数据契约将在后续门禁中补充。

## 1. 背景与方案结论
央行 E 平台是一套轻量内容发布平台，包含微信公众号 H5、WEB 管理后台和统一业务服务端。公众通过公众号进入 H5，公开浏览政策内容、县域服务指引和金融产品；后台用户按所属单位和角色维护内容。

已确认采用融合方案：

- H5 首页使用移动端卡片结构，主色为 `#1556D1`。
- 微信公众号 H5 不展示平台名称横幅，页面最顶部直接展示一级 Tab。
- 一级 Tab 使用分段按钮，二级 Tab 使用居中胶囊按钮，乡村振兴三级 Tab 使用文字导航和底部短指示线。
- 内容详情使用正式、克制的长文阅读结构，附件独立下载。

### 2.3 全局布局
- H5 首页无独立头部横幅，最顶部直接进入一级 Tab。
